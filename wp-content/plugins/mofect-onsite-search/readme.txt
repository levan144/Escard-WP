== Description ==

This plugin offers enhanced On-Site Search for your WooCommerce online shop, response the value-based products information when the user searching on your site, help you improve the sales conversion rate effectively.


== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory or install the plugin via Plugins page in WordPress backend.
2. Activate the plugin.
